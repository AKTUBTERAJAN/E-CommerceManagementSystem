# E-CommerceManagementSystem
BigMall is a full-stack e-commerce web application developed to provide a secure, fast, and user-friendly online shopping experience. The application enables customers to browse products, search by category, add items to the cart or wishlist, place orders, manage their accounts, and track their purchases.

The frontend of the application is built using HTML, CSS, JavaScript, React.js, and Vite, creating a responsive and interactive user interface. The backend is developed with Node.js and Express.js, which handle business logic, RESTful APIs, user authentication, and order management. MongoDB is used as the database to securely store user information, product details, orders, addresses, and other application data.

The project includes essential e-commerce features such as user registration and login, product management, shopping cart, wishlist, secure authentication, order tracking, customer support, profile management, and responsive design. BigMall demonstrates modern full-stack web development practices by integrating a React-based frontend with a Node.js, Express.js, and MongoDB backend to deliver a complete online shopping platform.
